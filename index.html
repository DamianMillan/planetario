Description

About the Planetarium 3D Project

The 'planetarium-3d' project is an interactive educational tool that allows users to explore the solar system in 3D. It uses NASA APIs to accurately position planets, asteroids, and comets, creating a visually attractive and realistic space simulation.

Users can navigate through the solar system, zoom into different celestial bodies, and learn precise and updated details about each one of them.

Additionally, the simulation presents realistic orbits and proportions that offer a deeper understanding of the movement and structure of the solar system. It's ideal for students and anyone interested in astronomy, providing an immersive experience that makes learning about the cosmos exciting and accessible.
